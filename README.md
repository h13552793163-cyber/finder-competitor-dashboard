# Finder Competitor Dashboard

## Overview

This project is an AI-powered competitor benchmarking dashboard developed for Finder.

It helps marketing and business teams monitor competitors, analyze social media content, and generate AI-powered business insights.

---

## Features

- Competitor benchmarking
- AI-generated summaries
- Marketing insight generation
- Dashboard visualization
- Automated competitor analysis

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- OpenAI API
